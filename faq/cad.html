FAQ／CAD設定

AutoCADの既定値画層（レイヤ）名"0"をNCVCに読み込ませたい NCVCの切削対象データは，既定値 "CAM" となっていますが簡単に変更可能です．
メニューの［オプション］→［CADデータの読み込み設定］で切削レイヤを以下のように設定してください．
CAM|^[0-9]$　CAM または 数字(0～9)で始まり数字で終わる という意味です．
切削レイヤの設定には正規表現が使えますので，文字のマッチングパターンを柔軟に表現可能です．
EASY DRAWでNCVC用にDXF出力時のレイヤ名を設定する方法 EASY DRAWでレイヤ（画層）に名前を付けることはできません．
EASY DRAWのデータ形式をDXFのデータ形式に変換するときに，レイヤに名前を付けることができます．
DXFコンバータを起動し，設定(S)→パラメータ設定(P)→レイヤ―設定(A)→DXF画層名称を追加(A) します．
NCVCEASY DRAW
レイヤ番号		DXF画層名称
切削レイヤ2CAM0
原点レイヤ3ORIGIN
加工開始位置指示レイヤ4START
強制移動指示レイヤ5MOVE
切削レイヤ6CAM1
切削レイヤ7CAM2
設定後，変換(C)
EASY DRAW→DXF(2)を実行すればOKのはずです．
注）作図した各軌跡を各レイヤに移動しておくことを忘れずに．
詳しくは，リファレンスマニュアルを参照してください．DXFコンバータの初期設定等は記述しませんでした．

by こいん様

＜＜NCVC側で対処する場合＞＞
EASY DRAWが吐いたDXFを解析すると，"CSLAYERn" という名前が強制的に付けられるので，NCVC側で
原点レイヤCSLAYER0
切削レイヤCSLAYER[1-9]
のような設定でも対応できます．
AdvanceCADでNCVC用にDXF出力時のレイヤ名を設定する方法 https://www.ctc-g.co.jp/solutions/acad/pdf/v19/DXF_DWG.pdf の "3.4 クラス／ピクチャ 変換テーブルファイル" に記載があります． ACADDXF_LAYER というファイルにメモ帳などのエディタで

CLASS 100=ORIGIN
CLASS 101=CAM1
CLASS 102=CAM2
...

と書き，AdvanceCADのクラス番号にDXF出力時の名前を割り当てればOKです．
※この情報はVer19時点(2012/06/28)のもので現在は他にやり方があるかもしれません．
