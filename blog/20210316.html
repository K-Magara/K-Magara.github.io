退院しました～2021/03/16

１．はじめに

　齢50を過ぎるとカラダのどこかにガタがくるもの． 2021年2月15日から3月16日まで30日間，人生初の入院生活を送っていました． 単に報告だけなら，わざわざページを作りませんでしたが， 振り返ると虫の知らせというか神の啓示にも似たような出来事の連続でした． 自分の記録用に入院までの経緯を書き留めておきます．

２．禁煙

　実は入院1年前の2020年2月17日から禁煙していました． 禁煙なんぞ何の自慢にもなりませんが，もし喫煙者のまま入院してたらと思うとゾッとしますね（笑）． 別に健康志向とかそういうので禁煙したわけではありません．むしろ ”吸えているほうが健康だ” と思っていたくらい（いまは真逆）ですが， もしかすると入院に備えての虫の知らせだったような気がします．ちょっとオーバーかな？

３．GitHubの本格利用

　前々から興味はあったんですが，単に食わず嫌いというか別に使わなくてもいいぢゃんという程度でした． そんな中 2020年4月 GitHub のプライベートリポジトリに関する規制が大きく緩和されたというニュースを知り，試したところこれが大ハマリ． 実験担当の相方と相談したところぜひやりましょうということで， 2020年の夏くらいから実験実習の指導書（授業レジュメ）を Git 管理下に置き，本格的に GitHub を利用することになりました． 指導書は TeX で作っているので Git との相性も抜群です． おかげで 2021年度の準備を入院病棟からリモートでできました．なんてすばらしい！！ これも2020年で本格利用を開始していないと，今ごろ相方さんにすごく迷惑をかけていたのかもしれません． まさにグッドタイミングでした． 別に入院中に仕事しなくてもいいと思うんだけど（笑）

４．ホームページの移設

　旧サーバ s-gikan2 の管理に限界を感じ移設先を探しておりましたが，別に問題なく動いているものをわざわざ移設するとなると， よしやるぞ的なそれなりの決意か，なんらかの事変が必要です． 前者は，上記 GitHub の本格利用をきっかけに，2021年の年明け早々から GitHub Pages の調査と移設準備を開始． 後者は，どこかで見ていたかのようなこのタイミングで，他高専での情報インシデント発生とサーバ管理者の利用点検通達． 今回の場合はその両方が重なり，1月で一気にサーバ移転を進め，2月1日に新装オープンとなった次第です． まるで神の啓示に導かれるかのように（笑）． あとは旧サーバをいつまで残すかを思案中でしたが，勢いのまま2月10日に一度閉鎖という形を取りました． のちに ”過去のサポート掲示板が見たい” と連絡が来たときは，ちょっと冷や汗かきましたけど（笑）．
　サーバ移設における唯一の失敗は，301リダイレクトを利用していなかったことです．あとになって気が付きました． 20年以上自前サーバを運営し，Googleさんにも十分認知されていたNCVCのページは，検索データベースのランクを自ら壊してしまったかもしれません． 復帰後まずやることは旧サーバに301リダイレクトを仕込むことかな？

　で，実はこの2月あたまから体調が思わしくなかったのです．

５．いまからはじめるNC工作第2版　重版連絡

　重版の連絡が出版社さんから来て，書類を送り返したのは2月12日の金曜日． タイミングが本当に神がかってましたね．締切はまだ先でしたが早めに出しておいて本当によかった． そして，その12日はあまりにも体調が悪く，仕事帰りに町医者を受診すると相当悪いことが判明． そのまま大きい病院に紹介状を書くと言われましたが，14日の日曜日に大事な約束があって意地でも帰りたかったので，そのままいったん帰宅しました．
　週が明けて2月15日の月曜日．その前の晩から症状がヤバかったので仕事を休んで朝から町医者を受信すると，そのまま大きい病院への入院手続き一直線でした．

以上が入院までの経緯となります．

６．まとめ

　2020年は自分でも不思議に思うくらい身辺整理的な作業を無意識に続けていた感じです． ”明日やろう”，”今度やろう” ではなく ”今やるべきことをしっかりやりきる” ことが大事だなと改めて感じさせられる入院でした．
　最後にもう1つミラクルを挙げるとすれば，一番授業に影響のない時期に症状が出たのも大きかったですね． 手術もなく薬が良く効いて本当によかったと思います．

追伸
禁煙して1年で10kg太りましたが，入院1か月で10kg痩せましたｗｗ
