Kodatuno CAM Kernel IGES

Kodatuno勝手に支援プロジェクト（仮）

プロジェクトの説明

　本プロジェクトは 金沢大学マンマシン研究室で開発されている オープンソースなCAMカーネル Kodatuno をもっと世の中に広めるため， 機能向上や使いやすさの向上（導入のしやすさ）を支援するものです． ベースとなるのは NCVC Ver4.00 で実装したNURBS曲面の対応，つまり Kodatuno とのリンクで得られた知見です． 本プロジェクトを推進することで，以下のメリットがあると考えます．

  • 自身の開発するアプリケーションにライブラリとして簡単に取り込むことができる
  • IGESファイルの対応
  • Kodatunoへの貢献==国産CAMカーネルへの貢献
　機械屋さんと情報屋さんがタッグを組んで，よりよいものを作れればと思います．

作業目標

　当面の目標として以下のことを考えています．

  • マルチプラットフォームを堅持しつつライブラリとして提供
    UIサンプルプログラムの切り離し，ヘッダーファイルの整理等々，CAMカーネルとして使いやすくするためのチューニング
  • モダンなC++コードへの置換
    STLコンテナを使用するなど先進的なC++コードの提供（UIサンプルの部分は従来通り）
  • IGESデータの対応拡充
    読めないIGESファイルへの対応
　VisualStudio用として作成したブランチも統合できればと思っています． オリジナル作者へのリスペクトも忘れずに．

GitHub Organization

https://github.com/KodatunoOrg

　以前は NCVC-CAM Organization 配下にKodatunoリポジトリを配置していましたが， 将来的にはこのOrganizationごとマンマシン研究室に譲渡できるよう新たにOrganizationを作り，その配下に移動させました． 今後 Kodatuno に関する活動はこの Organization 配下で行います． 将来的にURLが変更になることもご容赦ください．

Contributor募集

　Gitに精通していることが望ましいですが無くても大丈夫です． Issue報告やPullRequests歓迎です．
