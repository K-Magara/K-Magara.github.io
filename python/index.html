Pyton による NCプログラム（Gコード）の処理

はじめに

　本ページでは，NCVCの作者として 著書『いまからはじめるNC工作第2版』 で紹介している Perl スクリプトを Python で実現してみようという主旨で書いています． 習うより慣れろ！ 何を処理するか目的がはっきりしていればのみこみも早いでしょうし， それがなによりPython初心者向けの実践講座として有益だと思います．

徐々に増やしていきます．

NC関連

　以下のプログラムは GitHub に置いてあります． ご自由にどうぞ． 各スクリプトの [RAW] ボタンを押せば生スクリプトが取り出せますよ．

  • シーケンス番号を振る
  • ワードで分割してみる
    • →Scriptorium経由で実行する場合
  • 一括置換
  • Y軸の値をC軸の回転角度に変換
  • エピトロコイド曲線の座標を出力
  • スプライン曲線
  • Python版プロセスシート

グラフ

　本のサンプルだけではネタ切れなので，交流波形観測実験の指導書を書き直したときのサンプルプログラムを掲載します． 以下のプログラムは GitHub に置いてあります． ご自由にどうぞ．

  • ベース
  • 電圧の測定
  • 周波数の測定
  • 時間基準周波数測定法の説明グラフ
  • RL直列回路（誘導回路）
  • RC直列回路（容量回路）

　普通のグラフサンプルも

  • 理論値と測定値がある一般的な実験のグラフサンプル

その他

  • 複数のエクセルファイルを1つにまとめる
  • PythonでPIV解析
  • GyroBoyでライントレース（LEGO MindStorm EV3 MicroPython）
