Python プロセスシート（帳票作成）

Python版プロセスシート（帳票作成マクロ）

　著書『いまからはじめるNC工作 第2版』p.47 図2.25 で少しだけ触れています． 実際，Excelで作られた 帳票作成マクロ（ページ一番下） もダウンロードできるようにしてありますが， Excelのマクロって社内などの仲間うちで使うには良いんですが，ネットからダウンロードするマクロって怖いイメージがありますよね． これをPythonで作ってみました．ソースがオープンになっているので，安心して使えると思います． ただし，印刷用のヘッダーなど飾り付けはまだされていません． 


import sys
import os
import re
import pandas as pd

## 使い方
## python procsheet.py sample.ncd [sample.xlsx]
## 最後の出力ファイル名は省略可．省略すると 入力ファイル.xlsx で出力されます

## 引数処理
inFile=open(sys.argv[1])
outFile=os.path.splitext(sys.argv[2 if len(sys.argv)>2 else 1])[0]+'.xlsx'
print(outFile)

## 暗号みたいな正規表現は各自解析してください (^o^)
reg_ignore = re.compile(r'^(%|\()')
reg_word   = re.compile(r'[A-Z/]-?\d*\.?\d*')

## プロセスシートの列をDataFrameで作成
col = ['/','N','G','X','Y','Z','R/I','J','K','F','S','T','M','H/D','L','P','Q']
df = pd.DataFrame(index=[], columns=col)

## 1行（ブロック）ずつ処理
for line in inFile.readlines():
    ## コメント行は無視
    if re.match(reg_ignore, line):
        continue
    ## ブロックをワードごとに分割
    dic = {}
    wordlist = re.findall(reg_word, line)
    for word in wordlist:
        ## ワードの先頭1文字をキーに各列に文字列を追加
        w = word[:1]
        if w=='R' or w=='I':
            w = 'R/I'
        elif w=='H' or w=='D':
            w = 'H/D'
        dic[w] = (dic.get(w) or '') + word
    ## DataFrameに1行追加
    df = df.append(dic, ignore_index=True)

print(df)
df.to_excel(outFile, index=False)

procsheet.py

これを実行すると，以下のようなExcelファイルが出力されます．


暗号のような正規表現のところだけ解説を加えます．
まず reg_ignore = re.compile(r'^(%|\()') のところ．

  • '^'は，'～で始まる行'を表します．
  • '^'に続くカッコの中を見ると，'%|(' になっています．カッコは特殊文字のため'\('と書きます．つまり'%'か'('で始まる行ということになります．
  • reg_ignoreとマッチしたらcontinue，何もせずループを繰り返すということになります．

次に reg_word = re.compile(r'[A-Z/]-?\d*\.?\d*') のところ．

  • []は，'その中身のどれか' にマッチします．A-Z はそのままですね．'/' も入っているので，大文字アルファベットか'/'にマッチします．
  • 続く -? は，マイナス符号が0回または1回出現すればマッチします．
  • \d* は，0から9の数字が0回以上出現でマッチします．本当は \d+（数字が1回以上出現）と書きたいのですが，'/' オプショナルブロックスキップでは数字が省略される場合があるので \d* と書いています．
  • \.? は，小数点にマッチです．ドットもカッコ同様に特殊文字のため'\.'と書きます．? は0回か1回にマッチでしたから，小数点があってもなくてもマッチします．
  • 小数点以下の数値は省略される場合があるので，\d* でマッチします．

これで1つのワードにマッチします．

あとは煮るなり焼くなり好きに使ってください．NCVC本体インストーラに同梱予定です．
