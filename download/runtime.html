ランタイムライブラリ

基本的にインストーラに含まれています
MFCランタイム
64ビット版		 Visual Studio 2015 対応 MFCランタイム64ビット版 （14.0.24215.1）
MFCランタイム
32ビット版		 Visual Studio 2015 対応 MFCランタイム32ビット版 （14.0.24215.1）

※MFCランタイムが入っているはずなのに「インストールしますか？」と聞かれる場合

＜＜＜原因＞＞＞ インストーラで指示したキーが見つからないか違う．
32ビットの場合
　HKEY_CLASSES_ROOT\Installer\Dependencies\Microsoft.VS.VC_RuntimeMinimumVSU_x86,v14が
　{2E72FA1F-BADB-4337-B8AE-F7C17EC57D1D}でない．
64ビットの場合
　HKEY_CLASSES_ROOT\Installer\Dependencies\Microsoft.VS.VC_RuntimeMinimumVSU_amd64,v14が
　{7DC387B8-E6A2-480C-8EF9-A6E51AE81C19}でない．
Windows Update されていないか，違うものが入っているのが原因だが，多少バージョンが違っても動く場合があります．

＜＜＜対処方法＞＞＞
・「インストールしますか？」の問いに，いったん［はい］を選択．
・MFCランタイムのセットアップ画面が表示されるので，［インストール］ボタンがあればインストールする．
・［修復］［アンインストール］になっていれば［閉じる］を選択．この場合はすでにインストールされているものが使えます．

インストーラ側でももう少しチェック方法がないか模索中です．
